Let's Start

Τα μαθήματα μας για το εξάμηνο!

Καλώς ήρθατε στην καρδιά του εξαμήνου μας! Παρακάτω θα περάσουμε από τα 3 μαθήματα μας για αυτό το εξάμηνο.

Information Systems Modelling and Design (CN4000_1)

Καθηγητής: Κοσμίδης Παναγιώτης

Ώρες Μαθήματος: Τρίτη, 9:10 - 13:00

Ετοιμαστείτε να βουτήξετε στον κόσμο των διαγραμμάτων UML και του σχεδιασμού συστημάτων! Σε αυτό το μάθημα, θα μάθουμε πώς να μοντελοποιούμε και να σχεδιάζουμε πληροφοριακά συστήματα από το μηδέν, κυριαρχώντας στην τέχνη του να μετατρέπουμε τις επιχειρηματικές απαιτήσεις σε στιβαρά, επεκτάσιμα λογισμικά συστήματα. Αν σας αρέσει να λύνεις προβλήματα και να σχεδιάζεις τη ραχοκοκαλιά των επιχειρήσεων, αυτό το μάθημα είναι για εσάς. Φέρτε τα καπέλα σκέψης σας και τη δημιουργικότητά σας!

Computer Systems and Networks (CN4002_1)

Καθηγητής: Αρχοντής Αντώνης

Ώρες Μαθήματος: Τετάρτη, 15:10 - 18:00

Ανακαλύψτε τον κόσμο των υπολογιστικών συστημάτων και των δικτύων! Σε αυτό το μάθημα, θα μάθετε τα πάντα για τη δομή των υπολογιστών, την αρχιτεκτονική τους και πώς επικοινωνούν μεταξύ τους μέσω δικτύων. Από τις διευθύνσεις IP μέχρι τα πρωτόκολλα επικοινωνίας και τις καλωδιώσεις δικτύου, θα αποκτήσετε τις γνώσεις που απαιτούνται για να κατανοήσετε τον ψηφιακό κόσμο στον οποίο ζούμε. Αν αγαπάτε τα bits, τα bytes και τις επαναστατικές τεχνολογίες δικτύωσης, αυτό το μάθημα είναι για εσάς!

Web Technologies (CN4003_1)

Καθηγητής: Παπαδοπούλου Χρυσούλα

Ώρες Μαθήματος: Πέμπτη, 8:30 - 12:00

Ετοιμαστείτε να μάθετε όλα τα βασικά για τον κόσμο του web development! Σε αυτό το μάθημα, θα γνωρίσετε τις τελευταίες τεχνολογίες web, όπως το HTML, το CSS, το JavaScript και τις βάσεις δεδομένων. Από την κατασκευή ιστοσελίδων μέχρι τη δημιουργία διαδραστικών εφαρμογών, το μάθημα αυτό θα σας δώσει όλα τα εργαλεία για να δημιουργήσετε τη δική σας online παρουσία. Αν σας συναρπάζει το να φτιάξετε ιστοσελίδες που συνδέουν τον κόσμο, αυτό το μάθημα θα σας εξοπλίσει με τις γνώσεις και τις δεξιότητες για να κάνετε τα όνειρά σας πραγματικότητα.
