Web Technologies (CN4003_1)

General

Καλώς ήρθατε στο μάθημα Web Technologies! Στην τάξη αυτή, θα εξερευνήσουμε τον κόσμο των τεχνολογιών του Παγκόσμιου Ιστού (Web Technologies) και τον ρόλο που παίζουν στη δημιουργία σύγχρονων, διαδραστικών και δυναμικών ιστοσελίδων και εφαρμογών. Το Διαδίκτυο έχει γίνει αναπόσπαστο κομμάτι της καθημερινότητάς μας, και οι τεχνολογίες που το υποστηρίζουν εξελίσσονται συνεχώς

Κατά τη διάρκεια του εξαμήνου, θα καλύψουμε βασικές και προχωρημένες έννοιες που σχετίζονται με την ανάπτυξη ιστοσελίδων, από τις θεμελιώδεις γλώσσες HTML, CSS και JavaScript, μέχρι πιο προχωρημένες τεχνολογίες, όπως frameworks και εργαλεία ανάπτυξης εφαρμογών Ιστού.

Το μάθημα θα περιλαμβάνει θεωρητικές διαλέξεις και πρακτικές ασκήσεις, ώστε να αποκτήσουμε μια ισχυρή βάση και να αναπτύξουμε πραγματικά έργα. Επιπλέον, θα εργαστούμε σε ένα μεγάλο project, το οποίο θα αποτελέσει μέρος της αξιολόγησής μας. Σε αυτήν την εργασία, θα έχουμε την ευκαιρία να εφαρμόσουμε τις γνώσεις μας και να δημιουργήσουμε μια πλήρη και λειτουργική διαδικτυακή εφαρμογή.

Ετοιμαστείτε για ένα συναρπαστικό ταξίδι στον κόσμο του Web Development, όπου οι ιδέες σας θα μετατραπούν σε πραγματικές εφαρμογές!

Τι θα μάθουμε;

  • HTML (HyperText Markup Language): Η γλώσσα που χρησιμοποιείται για τη δομή του περιεχομένου μιας ιστοσελίδας.
  • CSS (Cascading Style Sheets): Η γλώσσα που χρησιμοποιείται για τη μορφοποίηση και τον σχεδιασμό των ιστοσελίδων.
  • JavaScript: Η γλώσσα προγραμματισμού που προσθέτει διαδραστικότητα στις ιστοσελίδες.
  • Responsive Design: Τεχνικές για τη δημιουργία ιστοσελίδων που προσαρμόζονται σε διαφορετικές οθόνες και συσκευές.
  • Client-Server Architecture: Κατανόηση της επικοινωνίας μεταξύ του προγράμματος περιήγησης και του διακομιστή.

Week 1 - HTML

Τι είναι μια ιστοσελίδα;

Μια ιστοσελίδα είναι ένα σύνολο εγγράφων HTML που εμφανίζονται μέσω ενός προγράμματος περιήγησης (browser) όπως το Google Chrome, το Firefox ή το Edge. Οι ιστοσελίδες φιλοξενούνται σε διακομιστές (servers) και είναι προσβάσιμες μέσω των διευθύνσεων URL (Uniform Resource Locator).

Ρόλος της HTML και της CSS

  • HTML (HyperText Markup Language): Η γλώσσα που χρησιμοποιείται για τη δομή και το περιεχόμενο μιας ιστοσελίδας.

  • CSS (Cascading Style Sheets): Η γλώσσα που χρησιμοποιείται για τη μορφοποίηση και την εμφάνιση μιας ιστοσελίδας.

Δομή ενός αρχείου HTML

Κάθε αρχείο HTML ακολουθεί μια συγκεκριμένη δομή. Παρακάτω παρουσιάζεται το βασικό πρότυπο (boilerplate) ενός αρχείου HTML: 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Τίτλος της Σελίδας</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Καλώς ήρθατε</h1>
    <p>Αυτή είναι μια παράγραφος.</p>
</body>
</html>
